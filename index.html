Web Security

    How would you advise a potential client to secure his/her website from web security attacks? Note: the word count for your answer should not be greater than 250 words

    First of all, we need web security to protect the information on the world wide web from web security threats.

    Here are some web security threats you should look into and be aware of. Cross-site scripting otherwise known as XSS, SQL injection and Cross site request forgery also known as XSRF.

Other important web security practices:

    - Make sure to always use "HTTPS" and "HSTS" on your website.

    - Sanitization of data

    - Secure authorization and authentication. A good advise is to have a secure authorization and authentication. By ensuring that authorization and authentication on your site is secure, it will be hard to crack.

    - Use of a well-known third-party application. You might have to do som research to find out if the application is secure and trusted. For example, you can check how many downloads the application gets per week or month to see if its a well known and trusted application.

    - Better password management. Avoid using the same password everywhere. You should have to use marks like "!?#" in the password to make it strong and more secure.
